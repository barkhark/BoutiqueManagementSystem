# Boutique Management System

This project is developed as part of MCA Semester 2 coursework.

The aim of this project is to understand how a Java web application works using
Spring Boot and MySQL. It includes basic concepts like project setup, database
connection, and simple backend logic.

The project currently focuses on:
- Creating a Spring Boot project
- Connecting MySQL database using JDBC
- Creating database tables using MySQL Workbench
- Understanding backend structure and flow
- Using Git and GitHub for version control

Technologies used:
- Java
- Spring Boot
- MySQL
- JDBC
- Maven
- Git & GitHub

This project is created for learning purposes and will be extended further
according to syllabus requirements.
